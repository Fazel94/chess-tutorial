2. **ببینید** — اصول در عمل: [بازی اپرا](tutorials/opening-principles/06-opera-game.md) و [جوئوکو پیانو](tutorials/opening-principles/07-giuoco-piano.md)
3. **تمرین کنید** — [خط‌های گشایش را حرکت‌به‌حرکت بازی کنید](practice.md)
4. **بازی کنید** — [در برابر کامپیوتر](play.md)
