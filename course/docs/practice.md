# تمرین گشایش

حرکات کتاب را در یک گشایش پرکاربرد تمرین کن. حریف به‌صورت خودکار حرکت‌های اصلی را بازی می‌کند؛ تو باید پاسخ کتاب را پیدا کنی. اگر اشتباه کنی، حرکت درست به تو نشان داده می‌شود و می‌توانی دوباره تلاش کنی.
